# Java GUI Calculator

A fully functional desktop calculator application built with Java Swing, featuring a clean graphical interface for everyday arithmetic operations.

## Features

- Graphical user interface built with Java Swing
- Supports addition, subtraction, multiplication, and division
- Responsive button layout with keyboard-friendly design
- Lightweight — no external dependencies required

## Tech Stack

- **Language:** Java
- **UI Framework:** Java Swing (javax.swing)
- **IDE:** Eclipse

## Getting Started

### Prerequisites

- Java JDK 8 or above
- Any Java IDE (Eclipse recommended) or command line

### Run

1. Clone the repository
   ```
   git clone https://github.com/saptarshiroy2007-sketch/Java_Calculator.git
   ```
2. Open in Eclipse (or your preferred IDE)
3. Run `Calculator.java` inside `src/my_package`
